# Calculadora Python

Uma calculadora simples com as quatro operações básicas.

## Funções

- `somar(a, b)` — retorna a + b
- `subtrair(a, b)` — retorna a - b
- `multiplicar(a, b)` — retorna a * b
- `dividir(a, b)` — retorna a / b (lança erro se b for zero)

## Como usar

```python
from calculadora import somar, dividir

print(somar(2, 3))    # 5
print(dividir(10, 2)) # 5.0
```
